log4js日志的重要性

发表于2019-06-28|更新于2021-03-30|nodejs
|评论数:

log4js起步

官方实例包含了常用场景的配置

起源来自于需要持久化console，开始用的fs，然后发现实在是真的麻烦，终于知道一个持久化日志的好处了

基本使用，等级trace<debug<info<error<fatal

ALL < TRACE < DEBUG < INFO < WARN < ERROR < FATAL < MARK < OFF

一定要设置level,默认是不显示任何东西的

配置

主要包含两个大项appenders与categories,然后通过log4js.configure进行配置

appenders定义不同的日志输出模式，如console，file

categories定义多个日志实例，默认使用default

log4js.shutdown() 关闭日志

连接http,下面会产生一个中间件函数

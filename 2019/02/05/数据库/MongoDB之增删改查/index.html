删除

remove() 方法

如果不再需要 MongoDB 中存储的文档，可以通过删除命令将其永久删除。删除 MongoDB 集合中的数据可以使用 remove() 函数。

remove() 函数可以接受一个查询文档作为可选参数来有选择性地删除符合条件的文档。删除文档是永久性的，不能撤销，也不能恢复。因此，在执行 remove() 函数前最好先用 find() 命令来查看是否正确。

操作符

  1. $all操作符，数组中的值都必须包含才返回
  1. $elemMatch 满足所有的条件（$elemMatch作用于查询数组值中至少有一个能完全匹配所有的查询条件的文档）
  1. $regex 正则表达式

  1. $slice 截取数组中的元素

  2. $set 更新或新增

  3. $unset 删除字段

  4. $rename 重命名

  5. $inc 数字增加

  6. $mul 数字相乘

  7. $min 原有值与预设值比较保留更小的

  8. $max 原有值与预设值比较保留更大的

  9. $size 查找指定数组长度的元素db.test.find({tags:{$size:3})

  10. $addToSet 数组字段增加一个值并且不会重复 { $addToSet: { <field1>: <value1>, ... } }

  11. $each 可以将数组展开

  12. $pop 删除数组最后一个元素

  13. $pull 从数组字段删除特定元素（只删除一个）

  14. $pullAll 从数组字段删除所有元素

  15. $currentDate 插入当前日期 db.restaurants.update({ “name” : “Juni” },{ $currentDate: { “lastModified”: true }})

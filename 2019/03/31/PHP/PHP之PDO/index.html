PDO(PHP Data Object)数据库访问抽象层，统一各种数据库的访问

  1. 编码一致性
  2. 灵活性
  3. 高性能
  4. 面向对象

PDO安装与配置

  1. 配置PHP配置文件，开启相应扩展：extension = php_pdo.dll
  2. 开启相对应数据库扩展：extension = php_pdo_mysql.dll
  3. 可以通过phpinfo()查看信息

dart语言学习

语言类型

在 dart 中所有类型都是变量

  • dynamic 动态类型，指没有指定类型的变量的类型(主要用于编译为 js 存在)，与 Object 的区别就是在编译期间不检查

  • final 和 const : const 是编译时常量，final 是在运行时初始化

  • numbers

    • int ：整数值，其取值通常位于 -253 和 253 之间
    • double 64-bit ：(双精度) 浮点数，符合 IEEE 754 标准

  • string

  • boolean

  • list (也被称之为 array) var list = [1,2,3]

  • map：只能使用 map[name]的形式调用

  • set: 不重复的值的集合

  • rune (用于在字符串中表示 Unicode 字符，2.6 以后使用这个 characters) '\u{1f44f}'

  • symbol

函数相关

  • 命名参数 void say({String arg1 = 'asd', arg2 = 2, c}) {}
  • 可选参数 String say(String from, String msg, [String device]){}

    可选参数与命名参数都可以和必选参数混合使用，但是只能放在前面

  • 默认值有两种方式 =和: 建议使用=

    注意只有可选参数和命名参数才可以有默认值

  • 如果没有明确指定返回值， 函数体会被隐式的添加 return null; 语句
  • 同一个类，不同实例的相同名字方法 是不相等的 (也就是说实例方法不共用)
  • 函数声明：int foo(int num1, int num2)或者typedef Calculate = int Function(int num1, int num2)
  • 匿名函数(){print('hello');}
  • 箭头函数()=>print('j=hello') 箭头函数只能有一行代码，不能有多行代码（可以使用逗号模拟）

dart 特殊的操作符

  • ...?: 扩展了…运算符，避免为空时报错[...?list]

  • as：类型转换 str as String

  • is： 如果对象是指定的类型返回 True（类似 instanceOf）

  • is!： 如果对象是指定的类型返回 False

  • b ??= value; // 如果 b 是 null，则赋值给 b；
    // 如果不是 null，则 b 的值保持不变

  • expr1 ?? expr2 ： 如果 expr1 是 non-null， 返回 expr1 的值； 否则, 执行并返回 expr2 的值。

  • ?.： 条件成员访问 ：左边的操作对象不能为 null，例如 foo?.bar 如果 foo 为 null 则返回 null，否则返回 bar 成员

  • 级联运算符 (..)

构造函数

调用顺序如下

  • initializer list（初始化参数列表）
  • superclass’s no-arg constructor（超类的无名构造函数）
  • main class’s no-arg constructor（主类的无名构造函数

子类在继承父类时，必须调用父类的构造函数，默认使用无参默认构造函数，
如果调用其它构造函数，需要在冒号后显示调用

初始化列表可以做到默认值做不到的一些事

工厂函数

普通函数默认返回 this，工厂函数可以自定义返回的东西，工厂函数不能访问this

Implicit interfaces（隐式接口）

每个类都隐式的定义了一个包含所有实例成员的接口

即在dart中类也是接口，使用implements继承类时，必须实现父类的实例成员

包的加载

  • import ‘package:mylib/mylib.dart’; 直接导入包
  • import ‘package:lib2/lib2.dart’ as lib2; 指定名称
  • import ‘package:lib1/lib1.dart’ show foo; 只导入一部分
  • import ‘package:lib2/lib2.dart’ hide foo; 不导入某一部分
  • import ‘package:deferred/hello.dart’ deferred as hello; 延迟加载

部分导出的话，可以使用export，或者给私有类加上_, (感觉是一开始没有设计包的概念)

Dart 和其他面向对象语言不一样，Data 中没有 public private protected 这些访问修饰符合, 而是在变量前添加_表示是一个私有

Callable classes（可调用的类）

如果 Dart 类实现了 call() 函数，则这个类的实例可以当做方法来调用

Metadata（元数据）

使用元数据给你的代码添加其他额外信息。 元数据注解是以 @ 字符开头 常量(例如 deprecated)

  • @deprecated 表示废弃的方法
  • @proxy 代理
  • @override 重写
  • @required 必填项

还可以自定义注解

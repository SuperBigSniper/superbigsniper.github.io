基本命令

修改终端语言

locale命令列出当前支持的语言

修改语言

显示日期

date +%Y/%m/%d: 2015/05/29

date +%H:%M: 14:33

显示日历

cal: 显示当前月份
cal 2015: 显示指定年份日历
cal 7 2015: 显示指定年份与月份日历

开关机
  1. 将数据同步写入硬盘中的指令： sync
  2. 惯用的关机指令： shutdown
  3. 重新开机，关机： reboot, halt, poweroff
改变文件属性与权限
  • chgrp ：改变文件所属群组
  • chown ：改变文件拥有者
  • chmod ：改变文件的权限, SUID, SGID, SBIT等等的特性
复制
文件内容查阅
  • cat 由第一行开始显示文件内容
  • tac 从最后一行开始显示，可以看出 tac 是 cat 的倒着写！
  • nl 显示的时候，顺道输出行号！
  • more 一页一页的显示文件内容
  • less 与 more 类似，但是比 more 更好的是，他可以往前翻页！
  • head 只看头几行
  • tail 只看尾巴几行
  • od 以二进制的方式读取文件内容！
解压缩

tar -zcvf xxx.tar.gz 目录

tar -jcvf xxx.tar.gz 目录 bz2压缩

将目录添加到xxx.tar.gz

z参数表示gz压缩，v参数表示显示执行过程,f表示后面接文件名，c表示打包

tar -zxvf xxx.tar.gz 目录 解压缩

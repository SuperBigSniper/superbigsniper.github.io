类型元数据

在runtime包中，类型信息包含2-3层

  • 第一层就是公共头header（type）
  • 第二层就是各自不同类型信息的body
  • 第三层是只有自定义包才有的，用来存储包信息

公共头信息

例如: slice类型

第三层是

type xxx int 与 type xxx = int 区别

type xxx int只是取一个别名，类型元数据指向同一个

type xxx = int会新建一个类型，指向不同的类型元数据

空接口interface{}

空接口长下面这个样子

  • _type就指向类型元数据
  • data指向对应的值

一个空接口元素，最开始两个值都为nil

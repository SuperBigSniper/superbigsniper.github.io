docker基本命令

发表于2020-02-03|更新于2022-04-06|docker
|评论数:

升级工具

sudo apt-get update

安装 docker(centos)

yum -y install docker-ce / apt-get install docke-ce

启动 docker

systemctl start docker

开机启动 docker

systemctl enable docker

查看 docker 的信息

docker info

搜索镜像

docker search node

配置国内镜像地址（https://registry.docker-cn.com）

编辑文件vi /etc/docker/daemon.json添加下面内容

重启 docker 服务

sudo systemctl restart docker.service

给当前用户组添加权限

只有 root 用户和 docker 组的用户才可以访问 Docker 引擎的 Unix socket

首先建立 docker 用户组sudo groupadd docker

将当前用户加入 docker 组：sudo usermod -aG docker $USER

拉取镜像(不写目录默认从 library 中拉去，也就是官方)

docker pull hello-world 拉取 hello-world

docker pull node 拉取 node 镜像

运行镜像

docker run hello-world

查看安装镜像的详细信息

docker image ls 或者 docker images

删除 docker

docker rmi hello-world 删除镜像

docker rm 容器 id 删除容器

docker rmi -r $(dcoker images -aq) 删除所有镜像

docker rmi $(docker images | grep “none” | awk ‘{print $3}’) 删除所有为none的镜像

当前运行的 docker 容器实例

docker ps

docker ps -a 查看运行过的容器

关闭 docker 容器实例

docker kill 容器id 强制停止

docker stop 容器id

docker container rm id

docker container prune 删除所有已经停止的容器

docker 后台运行(-p 是指端口映射将本机 8000 映射到 nginx 的 80 端口)

docker run -d -p 8000:80 nginx

docker run -d -P nginx 当使用 -P 标记时，Docker 会随机映射一个 49000~49900 的端口到内部容器开放的网络端口

进入后台运行的 docker 实例(-it 是指进入容器的 shell 并打开一个终端)

docker exec -it 容器 id bash

从容器中拷贝文件

拷贝容器根目录文件 nginx.txt 到当前目录

docker cp 容器 id:/nginx.txt .

拷贝当前目录拷贝的容器根目录

docker cp ./nginx.txt 容器 id:/

提交镜像

docker commit -m”描述” -a “作者名” 容器 id 作者名/项目名:版本号

查看日志

查看指定时间后的日志，只显示最后 100 行：

$ docker logs -f -t --since="2018-02-08" --tail=100 CONTAINER_ID

查看最近 30 分钟的日志:

$ docker logs --since 30m CONTAINER_ID

查看某时间之后的日志：

$ docker logs -t --since="2018-02-08T13:23:37" CONTAINER_ID

查看某时间段日志：

$ docker logs -t --since="2018-02-08T13:23:37" --until "2018-02-09T12:23:37" CONTAINER_ID

容器备份

docker save -o 保存的位置 容器 id

容器 恢复

docker load -i 被保存的容器

导出快照

docker export可以导出容器快照

docker export 7691a814370e > ubuntu.tar

导入容器快照

可以使用 docker import 从容器快照文件中再导入为镜像，例如

cat ubuntu.tar | docker import - test/ubuntu:v1.0

也可以通过 URL 进行导入

docker import http://example.com/exampleimage.tgz example/image.repo

查看端口映射

docker port imagesId 查看 imagesId 所有映射地址

docker port images 5000 查看 5000 的端口映射地址

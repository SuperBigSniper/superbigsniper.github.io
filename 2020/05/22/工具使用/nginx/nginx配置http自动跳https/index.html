http 与 https 端口不同

端口不同，思路很简单，监听 http 端口跳转 https

简便写法将 ssl 写在端口后面来表示该端口是否为 https

同一个端口下监听 http 与 https

思路就是通过 error_page 进行跳转，http 访问 https 页面会产生 497 错误

error_page 497 https://$server_name:$server_port;

Javascript之函数式编程

发表于2020-07-21|更新于2022-04-07|JS语法学习
|评论数:

引用透明性

函数对于相同的输入返回相同的值就被称为引用透明性

高阶函数

定义：高阶函数其实就是接受一个函数或者返回一个函数（或者即返回接受函数又返回函数）的函数

纯函数

定义：纯函数对于给定的输入返回相同的输出的函数（不会受到外界环境的影响，其实就是没有引用外部变量且没有改变外部变量）

优点：

  • 方便测试（测试用例方便填写）
  • 方便组合（小的 功能组合成大的功能）
  • 方便缓存

memoized（不是momorized）

可以缓存之前计算的结果(类似动态规划)

闭包

闭包其实并不算在函数式编程的定义里，但是又因为这个性质非常重要

定义：闭包就是在一个函数内的另一个函数

  • 函数里的函数叫做闭包函数
  • 闭包函数可以访问外部函数的变量（在将闭包函数返回出去也能访问！）

综上所述 其实就是闭包的概念了

柯里化

定义：柯里化就是把一个多参函数转化为一个嵌套的一元函数的过程

  • 可以把一个多参函数通过柯里化应用转化为具有特定功能的一元函数

偏应用

定义：偏应用其实是为了解决柯里化的不足，因为柯里化参数列表是从左到右，比如 setTimeout 就不能这样使用

组合(mixin)compose

定义：组合就是将小的函数，每个函数完成自己的功能，最后组合在一起完成一个大功能，react 中所有的东西都是组件，小组件组合成为大组件，也是这个思想

functor(函子)

定义：函子时一个普通对象或者类，它实现了 map 函数，在遍历每个对象值的时候生成一个新对象（一般还会实现一个of静态方法帮助我们快速生成一个函子）

  • 函子是一个容器
  • 函子实现了 map 方法（map 接受一个函数，将该函子的值交给这个函数运行，然后根据函数得到的值返回一个新的函子）

Maybe 函子

Maybe 函子是指在 map 方法中增加了对空值的判断的函子。

Either 函子

Either 函子是指内部有分别有左值（left）和右值（right），正常情况下会使用右值，而当右值不存在的时候会使用左值的函子。

Monad(单子)

Monad 函子是指能够将函子多层嵌套解除的函子。

我们往函子传入的值不仅仅可以是普通的数据类型，也可以是其它函子，当往函子内部传其它函子的时候，则会出现函子的多层嵌套

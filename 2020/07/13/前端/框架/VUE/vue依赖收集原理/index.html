vue依赖收集原理

发表于2020-07-13|更新于2022-04-06|VUE
|评论数:

简化版

依赖收集

有几个重要的对象

  • Vue构造函数，负责初始化
  • Observe负责对 data 中的数据进行劫持
  • Compile对模板进行编译，将有{{}}和指令的节点进行编译并进行依赖收集（将更新方法传入 Watch,这样之后就可以调用）
  • Watcher负责存入更新方法，一个节点更新就需要一个 watcher
  • Dep因为一个数据可能在多个节点使用，所以一个数据（data）会有多个 Watch,将 Watcher 存入 Dep，这样设置 data 的时候，直接将对应 dep 中的所有 watcher 触发即可

控制剪切板

发表于2020-07-02|更新于2022-04-06|浏览器端
|评论数:

clipboard.js

clipboard.js是一个实现自动复制的插件，通过select对元素进行选择，然后进行一系列封装

获取用户选择的内容

  1. 如果是input或者textarea元素
    • inputDom.focus();（inputDom.select()）
    • inputDom.setSelectionRange(2,5); // 选择特定部分
    • inputDom.setSelectionRange(0, -1); // 全选

最后执行document.execCommand(‘copy’)将内容复制到剪切板

程序主动选择

html5控制剪切板内容

主要有三个事件copy,paste,cut,这三个事件都会返回 clipboardData,通过clipboardData。getData('text')获取剪切板中的数据，通过e.clipboardData.setData('text/plain', '新数据')设置新的剪切板内容

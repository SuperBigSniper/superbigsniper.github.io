配置热加载

使用go mod管理依赖

下载 air

然后添加alias air='~/go/bin/air'这个命令

然后在项目目录下运行air即可

初识 gin

下载 gin

起一个 http

其中gin.H是一个帮助方法，定义如下type H map[string]interface{}

添加静态文件夹

加上这句话就好了app.Static("/static", "./tmp")，前面是虚拟目录，后面是实际目录

路由

直接使用app.GET就可以建立一个路由

路由分组需要使用app.Group就可以分组

获取参数

非 json 数据,可以直接获取，json 需要定义结构体

Query获取表单格式数据, 当一个参数有多个值，QueryArray, QueryMap获取数据
PostForm获取 body 数据, 当一个参数有多个值，PostFormArray, PostFormMap获取数据

POST /post?ids[a]=1234&ids[b]=hello 就可以使用QueryMap

下面是通过非json的方式获取数据, 包含get与post

默认会区分大小写

json 数据绑定通过ShouldBind进行绑定(ShouldBind会自动判断类型去绑定，比如BindJson)，可以指定form,json,uri,xml

ShouldBind会自动判断传输参数的类型，自动选择ShouldBindQuery, ShouldBind

还有一种比较流行的，通过路径获取参数，通过Param获取单个参数，通过Params获取所有参数

自定义验证

注册并使用一个验证器

/bookable?name=无字天书1这样的参数就会触发错误

返回消息为Key: 'Booking.Name' Error:Field validation for 'Name' failed on the 'mustBook' tag

validator也支持国际化验证器

返回数据

可以通过JSON,XML,YAML,PostForm,WriteString等方法返回数据

还可以返回html，不过需要使用LoadHTMLGlob设置目录

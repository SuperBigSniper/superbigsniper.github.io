go学习笔记

发表于2020-12-05|更新于2022-04-06|Go
|评论数:

package

  1. 首字母大写表示外部可以访问，方法，结构体，结构体内部字段
  2. import会先初始化全局变量，然后运行init函数（init可以与多个从前到后依次执行，多个文件按照名字的顺序执行），然后运行main函数
  3. 一个文件夹下面直接包含的文件只能归属一个package，同样一个package的文件不能在多个文件夹下。
  4. 包名可以不和文件夹的名字一样，包名不能包含 - 符号。
  5. 包名为 main 的包为应用程序的入口包，这种包编译后会得到一个可执行文件，而编译不包含 main 包的源代码则不会得到可执行文件。

私有变量和公有变量通过变量名大小写进行区分，大写为公有变量

nil

类似其它语言的null,在go中表示 指针、通道、函数、接口、映射或切片的零值

也就是说，在这几个类型中需要进行地址分配

这个 interface 比较特殊，interface 本质上是由两个部分组成type与value，只有两个值都为空，interfac 才为空

变量

go 有几种方法进行变量赋值

  1. var a int = 1
  2. var a = 1
  3. a := 1
  4. 还可以进行多个变量赋值简写
  5. _作为特殊符号，占位使用 a,_ = [1,2]

    全局变量必须使用var,局部变量不可以使用var

  6. 变量默认会有初始值

循环

go 语言只有for关键字作为循环

  1. 普通 for 循环
  1. while 循环
  1. 无限循环

switch

switch默认不会往下走，也就是不用break,不过可以通过fallthrough让程序继续往下走

运算符

按位清零符号&^，

  1. 如果右侧是 0，则左侧数保持不变
  2. 如果右侧是 1，则左侧数一定清零

数据类型

  1. int 与平台相关
  2. uint8 对应 byte
  3. int8 int16 int32 int64
  4. complex64 complex128 复数
  5. float32
  6. float64
  7. rune(int32 的同义词) unicode 编码类型，处理中文之类的类型，for range可以按照类型变量，普通for循环按照字节遍历

遍历带中文的字符串推荐使用for range，会自动帮你把每个字符byte转化为int32(rune)存储

流程控制

switch默认不会向下执行，加上fallthrough就可以匹配成功后也可以向下执行

数组与切片

  • 不知道数组中的值，也就是先声明不初始化var a [10]int
  • 知道数组中的值，也就是声明并初始化a := [10]int {}
  • 省略数组个数a := [...]int {1,2,3},
  • 指定某个下标的元素a := [20]int {19:1} 下标为 19 值为 1，其它默认为 0

和 python 一样，go 也有切片，arr[0:2]，注意a := []int {1,2,3}这种是一个切片，切片是引用类型，数组是值类型

切片的底层是一个数组，当进行append操作的时候会对底层数组进行改变, 如果容量未超过会对底层数组进行改变，如果容量超过会新建一个数组

还可以通过make初始化一个切片make([]int,3,10),长度为 3，容量为 10

panic

go 语言中，使用defer与recover捕获错误(panic)

map

切片与函数,map不能作为key, struct不包含以上类型也可以包含下面的

声明：a, ok := map[int]string{}或者a := make(map[int]string)

删除：delete(a, key)

make 和 new 的区别

  • 二者都是用来做内存分配的。
  • make 只用于 slice、map 以及 channel 的初始化，返回的还是这三个引用类型本身；
  • new 用于基本类型内存分配（很少使用一般是，var a int 就会对 a 进行内存分配了），可以在表达式中使用（和var a string取地址是一样的）而 new 用于类型的内存分配，并且内存对应的值为类型零值，返回的是指向类型的指针。

函数

函数使用func关键词进行声明

函数可以有可变参数a ...in(其实这个a就是个切片)

go没有默认参数，可选参数

go 的函数支持多返回值

需要注意的是多返回值必须都接受，可以用_去忽略

在函数名前面添加对应调用类型，就可以变成一个方法(就是将一个类型加强，可以让这个类型的实例调用方法)

go 提供延迟执行函数机制defer，函数将会从后向前调用,类似 js 的异步函数

defer的执行时机就是在函数结束的时候，并且是按照堆数据结构的顺序去执行

错误处理

panic与recover分别表示抛出错误与捕获错误

panic会停止当前函数执行，一直向上返回，执行每一层的defer

recover只能用在 defer 函数中, 任何错误都会被recover捕获

但是如果有多个匿名结构体且他们内部有相同的字段时，就不能直接访问了

unsafe

unsafe为我们提供了可以直接操作指针与内存的工具

  • unsafe.Alignof 查看对齐的边界值
  • unsafe.Sizeof 查看所占内存大小
  • unsafe.Offsetof 提供指针偏移
  • unsafe.Pointer 提供将安全指针转化为不安全指针

例如，你想要对指针进行偏移的操作，就可以使用到这个unsafe.Pointer, 他可以将指针转化为unsafe.Pointer，然后再转化为对应的非安全指针uintptr

unsafe.Pointer是一种特殊意义的指针，它可以转化为任意类型的指针，任意指针也可以转化为unsafe.Pointer

上面通过直接加8, 是因为我的计算机是64位，会有8个字节的偏移，如果是结构体，我们也可以使用unsafe.Offsetof()去动态获取偏移

整个unsafe包都是用于Go编译器的，不用运行时，在我们编译的时候，Go编译器已经把他们都处理了

反射 reflect

reflect.TypeOf与reflect.ValueOf

获取类型reflect.TypeOf

获取的类型变量还有,kind()与Name()方法，获取底层类型与名称

注意type myInt int64与type myInt = int64是不同的，后者是别名，无法添加方法，前者是拷贝可以添加方法

Go 语言的反射中像数组、切片、Map、指针等类型的变量，它们的.Name()都是返回空

通过reflect.ValueOf反射获取值，具体步骤先确定类型，然后确定值

并且ValueOf获取的值还可以通过Type获取reflect.TypeOf

想要在函数中通过反射修改变量的值，需要注意函数参数传递的是值拷贝，必须传递变量地址才能修改变量值。而反射中使用专有的Elem()方法来获取指针对应的值。

可以通过CanAddr()判断是否可以寻址，防止报错，但是不能确定是否可以修改，CanSet以检查对应的 reflect.Value 是否可取地址并可被修改

结构体反射

可以通过reflect.TypeOf(o).NumField()拿到所有字段数量

然后通过Field(i)取对应索引的字段

还可以通过FieldByName通过名字取到对应的字段

注意FieldByName限制较多，当你的tag不标准是无法取数据的

同理还可以通过反射获取对应方法,通过Method或MethodByName获取方法，通过call调用

调用时，参数是reflect.Value类型，使用reflect.ValueOf初始化参数

BOM&DOM常用API

发表于2020-10-08|更新于2022-04-06|浏览器端
|评论数:

BOM

  • screenleft：获取窗口距离屏幕左边的距离
  • screentop：获取窗口距离屏幕顶部的距离
  • open:打开新的窗口
  • close：关闭窗口
  • window.confirm()：弹出取消确定框
  • window.prompt()：弹出输入框
  • 事件类型 :
    • mouse：”mousedown”, “mouseup”, “click”, “dblclick”, “mousemove”, “mouseover”, “mouseout”, “mousewheel”
    • key：”keydown”, “keyup”, “keypress”, “textInput”
    • touch：”touchstart”, “touchmove”, “touchend”, “touchcancel”
    • control：”resize”, “scroll”, “zoom”, “focus”, “blur”, “select”, “change”, “submit”, “reset”

DOM

  • getAttribute(“属性名”,”内容”)
  • setAttribute(“属性名”,”内容”)
  • querySelector(“div”)：接受一个css选择符，返回第一个元素
  • querySelectorAll(“div”)：接受一个css选择符，返回所有的元素即一个数组
  • 元素.classlist.add：将类名添加
  • 元素.classlist.remove：将类名删除
  • 元素.classlist.toggle：将类名替换
  • 元素.classlist.contains：检查类名是否存在
  • 元素.style.css属性：获取或者设置css（设置的是内置样式表）
  • 元素.style.cssText：获取或者设置多个css（设置的是内置样式表）
  • 元素.offsetHeight：获取元素高度（包括边框）
  • 元素.offsetWidth：获取元素宽度（包括边框）
  • 元素.offsetLeft：获取边框距离左边的距离
  • 元素.offsetTop：获取边框距离上方的距离
  • 元素.clientHeight：获取元素高度（不包括边框）
  • 元素.clientWidth：获取元素高度（不包括边框）
  • 元素.scrollLeft：滚动条距离左边的像素
  • 元素.scrollTop：滚动条距离上方的像素
  • 页面滚动条距离：document.documentElement.scrollTop或者document.body.scrollTop
  • 元素.focus()：元素获得焦点
  • 元素.dataset.属性：获取自定义属性（如 data-myname = dataset.myname）
  • 元素.innerHTML：插入HTML元素
  • 节点.parentNode：父节点
  • 节点.nextSibling：紧邻在后的那个同级节点
  • 节点.previousSibling : 紧邻在前的那个同级节点
  • 节点.appendchild：追加节点到末尾
  • 节点.inserbefore：第一个参数是所要插入的节点newNode，第二个参数是父节点parentNode内部的一个子节点referenceNode。newNode将插在referenceNode这个子节点的前面。返回值是插入的新节点newNode
  • var chils= s.childNodes; //得到s的全部子节点（会包括换行，所以一般不用）
  • var chils= s.children; //得到s的全部子节点(不包括换行)
  • var par=s.parentNode; //得到s的父节点
  • var ns=s.nextSbiling; //获得s的下一个兄弟节点
  • var ps=s.previousSbiling; //得到s的上一个兄弟节点
  • var fc=s.firstChild; //获得s的第一个子节点
  • var lc=s.lastChile; //获得s的最后一个子节点
  • document.domain：获取和设置域名
  • element.getAttribute：获取元素属性
  • element.setAttribute：设置元素属性
  • document.createElement(“div”)：创建新元素
  • 事件
  • 元素.addEventListener(元素，function，冒泡or捕获（flase true）)：添加事件
  • 元素.removeEventListener(元素，function，冒泡or捕获（flase true）)：移除事件
  • 元素.dispatchEvent(event) ：触发事件，注意event是一个变量 let event = new Event(‘click’)
  • resize：窗口变化是触发事件
  • load：页面加载完成触发事件
  • unload：页面关闭时触发事件
  • beforeunload：在unload前触发
  • blur或者focusout：失去焦点时触发
  • focus或者focusin：得到焦点时触发
  • mouseenter：鼠标进入触发事件
  • mouseleave：鼠标离开触发事件
  • mousedown：鼠标按下触发
  • mouseup：鼠标抬起触发
  • onchange：text文本改变是触发
  • onscroll：滚动条改变时触发
  • click：单击触发
  • dbclick：双击触发
  • keyDown:键盘按下
  • keyUp:键盘弹起
  • keyPress:键盘按下（但是这个是按下字符键才触发）
  • mousewheel和DOMMouseScroll：鼠标滚轮事件（前者大多数浏览器，后者火狐）
  • invalid：用户提交表单时，如果表单元素的值不满足校验条件，就会触发invalid事件。
  • readystatechange : loading：加载 HTML 代码阶段（尚未完成解析）interactive：加载外部资源阶段complete：加载完成
  • DOMContentLoaded

event对象

  • event.currentTarget：执行事件的节点
  • event.target：触发事件的节点
  • event.peventDefault()：取消事件的默认行为
  • event.stopPropagation()或者event.cancelBubble：取消事件的冒泡或者捕获
  • event.stopImmediatePropagation() ：取消事件触发
  • event.type：触发事件的类型
  • event.key：返回一个字符串，表示按下的键名，如果按下的键代表可打印字符，则返回这个字符，比如数字、字母。
  • 如果按下的键代表不可打印的特殊字符，则返回预定义的键值，比如 Backspace，Tab，Enter，Shift，Control，Alt，CapsLock，Esc，Spacebar，PageUp，PageDown，End，Home，Left，Right，Up，Down，PrintScreen，Insert，Del，Win，F1～F12，NumLock，Scroll 等。
  • event.code：属性返回一个字符串，表示当前按下的键的字符串形式。该属性只读。
  • event.keyCode：按下的键的编码
  • KeyboardEvent.location: 属性返回一个整数，表示按下的键处在键盘的哪一个区域。它可能取以下值。
  • KeyboardEvent.getModifierState():方法返回一个布尔值，表示是否按下或激活指定的功能键。它的常用参数如下。
  • event.clientX：客户区鼠标水平位置
  • event.clientY：客户区鼠标垂直位置
  • event.pageX：整个页面鼠标水平位置
  • event.pageY：整个页面鼠标垂直位置
  • event.screenX：屏幕上鼠标水平位置
  • event.screenY：屏幕上鼠标垂直位置
  • MouseEvent.offsetX：属性返回鼠标位置与目标节点左侧的padding边缘的水平距离（单位像素）
  • MouseEvent.offsetY：属性返回与目标节点上方的padding边缘的垂直距离。这两个属性都是只读属性。
  • event.shiftKey：shift键按下
  • event.ctrlKey：ctrl键按下
  • event.altKey：alt键按下
  • event.metaKey：meta键按下
  • event.button：0代表左键，1代表中键，2代表右键
  • Event.isTrusted：属性返回一个布尔值，表示该事件是否由真实的用户行为产生
  • Event.detail ：对于click和dbclick事件，Event.detail是鼠标按下的次数（1表示单击，2表示双击，3表示三击）；对于鼠标滚轮事件，Event.detail是滚轮正向滚动的距离，负值就是负向滚动的距离，返回值总是3的倍数
  • Event.composedPath()：返回一个数组，成员是事件的最底层节点和依次冒泡经过的所有上层节点

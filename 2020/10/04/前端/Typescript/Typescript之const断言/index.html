as const

const

  • 用于变量；
  • 变量不能重新赋值给其他任何事物。

翻译一下，其实有的时候，有的类型长这样type right = 'right'
但是呢，有一个对象变量const obj = { str: 'right' }, 这个时候str的类型为string
和right的类型不兼容，所以需要使用const
const obj = { str: 'right' as const }

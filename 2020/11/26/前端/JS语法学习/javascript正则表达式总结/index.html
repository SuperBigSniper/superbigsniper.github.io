javascript正则表达式总结

发表于2020-11-26|更新于2022-04-06|JS语法学习JavaScript
|评论数:

声明一个正则表达式

一般有两种方式，一种是字面量，一种是构造函数

元字符

就是说在正则表达式中有一些特殊含义的字符

修饰符

常用的修饰符有：i,g,m 分别表示 忽略大小写，全局匹配，多行匹配

在新的es标准中增加了一些修饰符，s，u，y 分别表示doAll模式，匹配Unicode字符串，粘连sticky修饰符

doAoll模式就是: .默认不会匹配换行和四个字节的 UTF-16 字符,加上之后就可以匹配了

粘连sticky修饰符: 就是将lastindex每次都会从头开始匹配

贪婪与非贪婪模式

在量词匹配中，会出现{3,6}，表示3到6次，那到底是匹配3次还是匹配6次呢，所以贪婪模式就出来了，默认是贪婪模式，会匹配6次，6次不行再去匹配5，4，3 ，非贪婪就是说加上 ? 就是使其优先匹配 3次

捕获与具名捕获

捕获就是指使用了分组()，然后在表达式中可以使用这些分组，replace使用 $1，$2，$3……表示，其它方法保存在返回的数组中的 第1，2，3….个参数中

具名捕获就是指有名字的捕获，默认会放到groups，使用?<name>

先行断言与后行断言

断言就是指 匹配的字符串 前后的字符串是不是某个特定的字符串

正则表达式属性

新特性

  • 命名分组：(?\d{4,4}) 可以通过groups.year获取指定的分组
  • 后向断言：/\d+(?=%)/ 使用 ?<= 表示百分号前面的数字
  • 前向断言：?=

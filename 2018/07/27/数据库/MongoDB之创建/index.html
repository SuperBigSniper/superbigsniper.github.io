初认识

NoSql和传统的数据库有很大的区别，没有表，行列的概念，其中一种是 集合collection 和 文档document

而集合里面存的又是 JSON对象 的键值对

  • mongod –dbpath c:\mongo 开启数据库

  • mongo 连接数据库

  • show dbs 列出所有数据库

  • use “数据库名” 使用数据库（如果没有则新建数据库）

  • 插入数据 db.student.insert({“name”:”xiaoming”,”age”:12});

  • 显示集合 show collections

  • 查找student集合中的数据

    db.student.find()

    db.student.find({age:80})

  • 删除数据库
    db.dropDatabase()

  • 数据导入
    mongoimport –db itcast –collection student –drop –file c:\student.json

添加与删除服务

  1. 添加服务：

    mongod.exe –dbpath e:\mongo（数据库位置） –logpath E:\mlog –logappend(追加日志) –directoryperdb（每个数据库单独文件夹） –serviceName mongodb –install（安装为一个服务）

  2. 删除服务：

    sc delete mongodb

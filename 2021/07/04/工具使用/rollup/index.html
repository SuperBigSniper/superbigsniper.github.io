rollup是什么

Rollup 是一个 JavaScript 模块打包器，但是它不像webpack一样自己搞一个打包程序，而是支持像amd, cjs, system, es, iife or umd等等

而且rollup支持同时打包成多种模块, 只需要将output配置项配置为一个数组

rollup -c 就可以打包了, 默认使用 rollup.config.js

插件

rollup有一个非常重要的功能，那就是插件plugins

例如

rollup.js编译源码中的模块引用默认只支持 ES6+的模块方式import/export。然而大量的npm模块是基于CommonJS模块方式，这就导致了大量 npm 模块不能直接编译使用。所以辅助rollup.js编译支持 npm模块和CommonJS模块方式的插件就应运而生。

  • rollup-plugin-node-resolve 插件允许我们加载第三方模块
  • @rollup/plugin-commonjs 插件将它们转换为ES6版本

通过这两个插件，我们就可以将第三方的cjs编写的包引入

如果要使用typescript,可以使用@rollup/plugin-typescript

记得配置tsconfig

同理还有很多其他插件

  • rollup-plugin-postcss
  • rollup-plugin-serve // 开启dev Server
  • rollup-plugin-livereload // 配置热加载
  • @rollup/plugin-alias // 配置路径别名
  • rollup-plugin-copy // 复制文件
  • @rollup/plugin-image // 转为base64
  • @rollup/plugin-json // json转es6模块
  • rollup-plugin-visualizer // 可视化分析打包模块
  • rollup-plugin-web-worker-loader // 处理web-worker

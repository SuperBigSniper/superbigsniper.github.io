区块链概念

发表于2021-04-20|更新于2022-06-06|区块链
|评论数:

DeFi

全称decentralized finance，去中心化金融，称为“分布式金融”或者“开放金融”更合适

DeFi 是什么？它一般是指基于智能合约平台（例如以太坊）构建的加密资产、金融类智能合约以及协议

本质上表示一种概念，其实会有很多实现（dex，uniswap）

  • 不需要通过信任的金融机构

  • 智能合约是它的一部分

  • 大多数基于以太坊的公链（为什么要基于以太坊的公链，因为它最大？？既然是以太坊的公链为什么不直接使用以太坊要搞一个新的出来）

  • erc20 是什么? 是一种标准，因为有很多代币，类似一种 interface 约定，我们通过指定的函数签名等返回我们需要的东西

  • gas 为了防止别人无限运行无关合约，所以有这种 gas 机制，你运行智能合约需要使用 gas（天然气）才能运行，不同的命令需要不同的 gas

常见的defi实现

  • compound(uniswap 类似) 流动池交易 使用流动池进行借贷，想要放贷的人将资金存入（按照资金比例发放利息），借贷人使用 eth 等兑换代币（会有利息产生），缺点很明显你需要资产进行抵押
  • Dharma，dYdx（p2p 撮合模式，类似挂单，撮合借方和买方），放贷时资金会被锁定，但是只有被借出去才会有利息
  • MakerDao(DAI) 稳定币模式

应用场景

  1. 稳定币 DAI
  2. 借贷
  3. 交易所
  4. 衍生品
  5. 基金管理
  6. 彩票
  7. 支付
  8. 保险

uniswap

uniswap？基于这个公式 x*y = k，k 恒定不变，x 表示代币供应量，y 表示 eth 价格
本质是一组部署到以太坊网络的合约

这个 swap 价格或者说兑换比例是怎么来的呢？就是很多人都在进行 swap，根据大家兑换的价格进行浮动，兑换会比交易所更贵

pool 主要是收取手续费进行赚钱

  • swap 交换：（貌似不会通过交易所进行币与币之间的交换）如果一个币没有上交易所，那么就只能通过这种方式
  • pool 做市商 ：你提供对应的币种，然后你可以获得对等的手续费(0.02%)，把你的币存到 pool 里，然后根据池子里的币的比例获得利润

反应迟钝，所以 swap 与 pool 都不能保证赚钱

以太坊的代币

相当于是一种表示数字资产的智能合约，这就是以太坊代币

这种代币发放一般叫ico类似股市ipo，代币更多的是像一种股份，你的项目挣钱了，你手上的代币就会更加值钱，然后就会升职

问题？uniswap是怎么挣钱的？？？难道是用募集的eth去做pool??

具体交易大概是这样？ 我有 eth，然后我去购买别人的代币（其实就是运行别人的智能合约，然后合约扣除相应的 eth 然后在区块上记录信息）
然后我就完成的对这个代币的购买？

智能合约

在以太坊中，你可以创建一个包含代码的新帐户，并在交易中将其上传到以太坊的区块链上，从而建立一个智能合约

  • Solidity ——类似 Javascript 的语言，是目前最流行的、具有强大功能的智能合约的脚本语言。
  • Serpent ——类似 Python 的语言，在以太坊发展早期比较常用。
  • LLL——类似于 Lisp，仅用于较为早期的语言编写中。LLL 或许是最难编写的语言。

因为任何人都可以编写，所以这个智能合约的代码是存在区块链上的，通过的以太坊的vm进行运行,也就是EVM，类似 v8 功能

合约无法被修改，比如TheDAO被攻击事件，导致硬分叉,也就是后面的etc又叫ethereum classic

DAI

通过maker vault合约进行绑定

通过抵押债仓（CDP）进行DAI币的兑换，兑换之后，还债需要支付利息和本金

当你的抵押物不足以还债时会执行清算,比如你抵押的是eth然后这个eth突然暴涨，那么就会稀释你手上的CDP，你用同意的DAI就只能获取更少的ETH

钱包

eth 推荐mist

MetaMask浏览器钱包插件

公链，联盟链，私有链

  • 公链：所有人都可以访问
  • 联盟链：只有链的参与者的人才可以访问（ibm）
  • 私有链：私人链

账户（分为普通账户与合约账户），地址，交易（向空账户发送信息就可以自动以信息为代码块的合约账户）

truffle web3js

跨链：可通过资产跨链桥，将 ETH、BTC、稳定币等资产映射到 Heco，实现方式为在源链上锁定一定数量的资产后在 Heco 生成对应数量的 Token。 Heco 鼓励社区开发者提供更多去中心化的跨链解决方案

pow,pos,dpos

pow 工作量证明：根据谁的算力强，谁说了算
pos 权益证明：你的币越多并且持有币的时间越长（这个简称币年龄,数量*持有天数），你的 hash 算法难度越小（如果被发现作恶，那么就会损失持有的代币） - pos 允许分叉
dpos 分布式权益证明：相当于股东投票选出代理人，代理人进行验证和记账（例如EOS）,缺点：选举可能会被操作，过于中心化

以太坊运行在30303,运行一个叫Devp2p协议

以太坊 2。0

  • sharding 分片 分层
  • plasma 侧链

红色的就是侧链，以太坊 2.0 可以容纳不同的共识机制pos,pow等等，但是主链是采用pos机制对侧链进行投票

解决了主网运算能力集中的问题

侧链负责打包计算，完成后然后才给主链

信标链，需要 eth 转 Beth 且无法转回，三年内无法交易
